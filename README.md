<h2 align="left">Controls: </h2>

Player 1:
- Arrow keys to move
- Space bar to shoot

Player 2:
- WASD to move
- Shift to shoot


<h2 align="left">Tech Used: </h2>

- JavaScript
- HTML
- CSS
- SCSS
- Firebase
